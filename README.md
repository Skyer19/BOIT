# BOIT
yh蒟蒻写的给OIers用的一个bot，OI Bot的第二版。

# features
**小!!!**  目前可执行文件和dll打包大小不到300KB
 
**快!!!**  多线程架构并发处理指令，run等高性能要求指令使用IO完成端口负载均衡技术

**功能强大**  内置沙盒，可以运行代码，支持输入数据等功能，内置查询OIer等功能，查询codeforces等功能

**易于部署**  程序本身不需要任何依赖（除CoolQ），可轻易部署
